How to Use Word Doc Synth

REMINDER: NOTHING WILL PLAY IF . ISN'T USED!

Letter → single sustained frequency (default 1 second).

Word → chord (all letters played together once).

[ ... ]. → loop everything inside (updates live, stops when deleted).

! → louder for that note/chord only

? → random pitch

( ... ) → live mode: plays notes as soon as you type it (no chords)

→ extend sustain by +1 second.

; → shorten sustain by -0.5 seconds.

/ → silence (rest).

" ... " → muted (text exists but makes no sound).

. → plays everything in that paragraph (nothing will sound if this isn't typed)
